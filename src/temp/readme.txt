# What Happens Here?

## Step by Step Explaination : 

1. 	After the Script is inputted, a Simple Plain tts is Generated 
	with the Help of pyttsx3.

2.	Then Using the Previous plain_Temp.mp3, a new mp3 is created
	which is the final mp3 (containing all the Audio Effects) , 
	also the previous `plain_Temp.mp3` will be removed in this
	step.

3.	Now a Video is Generated with the Audio,Subtitles and with
	the respected Stock Video Clip.
